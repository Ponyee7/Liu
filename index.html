零基础使用GitHub Pages创建您的个人网站

什么是GitHub Pages？

GitHub Pages是GitHub提供的一项静态网站托管服务，可以免费托管个人、项目或组织的网站。您可以使用HTML、CSS和JavaScript直接创建网站，也可以通过静态网站生成器（如Jekyll）生成网站。

本教程将指导您如何使用GitHub Pages在几分钟内创建并发布您的第一个网站。

1

创建GitHub账户

如果您还没有GitHub账户，请先访问 GitHub官网 注册一个免费账户。

2

创建新仓库

登录GitHub后，点击右上角的"+"号，选择"New repository"。

仓库名称必须格式为：您的用户名.github.io

例如，如果您的用户名是"example"，则仓库名应为"example.github.io"。

3

添加网站内容

创建仓库后，点击"Add file" → "Create new file"。

文件名输入"index.html"，然后在编辑器中添加以下HTML代码：

<!DOCTYPE html> <html> <head> <title>我的网站</title> </head> <body> <h1>欢迎访问我的网站！</h1> <p>这是我的第一个GitHub Pages网站。</p> </body> </html>
4

发布网站

滚动到页面底部，点击"Commit new file"。

等待几分钟后，访问 https://您的用户名.github.io 即可查看您的网站。

进阶提示

  • 您可以使用自定义域名替换默认的github.io域名
  • 尝试使用Jekyll主题让网站更美观
  • 通过Git命令行工具可以更高效地管理网站内容
  • 使用GitHub的版本控制功能可以追踪您网站的所有更改
